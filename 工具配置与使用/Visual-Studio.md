># 快捷键使用
-  进入类、方法、变量
按住` Command` 键 + 点击某个变量或者方法

-  返回或进入某个方法
    - 进入：Command + option + ➡️
    - 返回：Command + option + ⬅️

># 快捷键修改

- 方法定义的进入和返回
    - 进入
        Windows：Ctrl + 鼠标左击跳到定义；
        Mac： Command + 鼠标点击；

    - 返回：
        Windows: Alt + ← ;或者 鼠标侧键
        Linux: Ctrl + Alt + - ;貌似数字键盘的减号没效果
        Mac: Ctrl + -
        终极奥义：VSCode左下角 “管理 / Manage” -> “键盘快捷方式 /     Keyboard Shortcuts” -> 搜索 “前进 / Go Forward 或 后退 / Go Back”

左右括号之间跳转：
ctrl + shift + \
